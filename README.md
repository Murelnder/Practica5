# Descripción:
Este es un programa donde se crea un mazo utilizando ArrayList para guardar una baraja de cartas de 52 cartas. Se utiliza otra clase llamado jugador que recibe 5 cartas y utilizando métodos para barajar cartas y que los jugadores reciben cartas y pueden regresarlas  al mazo.

-Carta- 
Esta es la clase para crear el objeto Carta.
- Mazo-
Esta es la clase donde se guarda los objetos Carta.

-Jugador-
Esta es la clase donde se crear el objeto jugador. 

-Juego-
Esta es la clase main donde corre el programa.

Para ejecutar el proyecto desde la carpeta raíz:
### En Git Bash
./gradlew run


# Instrucciones de ejecución
En la carpeta: app/src/main/java/practica5/

-Mazo.java
-Carta.java
-Jugador.java
-Juego.java

Aquí están los archivos para realizar las pruebas unitarias del programa.
